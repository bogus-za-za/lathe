|  | 切断刀 | 将工件从棒料上切断 | 韧性好的硬质合金、涂层刀片 | 专用切断刀杆+切断刀片 | 比切槽刀更长更突出，排屑槽设计关键 |

| **螺纹加工** | 外螺纹车刀 | 车削外螺纹 | 硬质合金、高速钢 | 螺纹刀杆+成型螺纹刀片 | 刀尖形状为螺纹牙型（60°三角形等） |
|  | 内螺纹车刀 | 车削内螺纹 | 硬质合金 | 内螺纹刀杆+成型螺纹刀片 | 刀杆细长，刀头为螺纹牙型 |
| **成型加工** | 仿形车刀/圆弧车刀 | 车削复杂轮廓形状 | 硬质合金 | 模块化刀杆+成型刀片 | 刀尖是完整圆弧（R刀片）或特定轮廓 |
|  | 倒角刀 | 加工孔口或轴端的倒角 | 硬质合金 | 模块化刀杆+专用倒角刀片 | 刀尖角度固定（如45°），多刃设计 |
