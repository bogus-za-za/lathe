### 车床宏程序（用户宏程序）全面解读
:::tip
宏程序 = 带**变量、运算、逻辑判断、循环**的高级 G 代码
替代手工编写重复段落，适合：**椭圆/抛物线/非圆曲线、梯形螺纹、变螺距、批量相似零件**
:::

| 项目 | 含义 | 说明 |
| :--- | :--- | :--- |
| **#** | 变量符号 | 宏程序核心，所有变量以 # 开头 |
| **G65** | 宏程序调用 | 调用独立的宏程序文件 |
| **IF / THEN** | 条件判断 | 如果满足条件，则执行动作 |
| **WHILE / END** | 循环指令 | 满足条件时循环执行 |
| **运算符号** | + - * / 等 | 变量之间数学计算 |

---

## 一、变量类型（必须掌握）
| 变量号 | 类型 | 用途 |
| :--- | :--- | :--- |
| **#1~#33** | 局部变量 | 程序内临时使用 |
| **#100~#149** | 公共变量 | 跨程序使用 |
| **#500~#509** | 保持变量 | 断电不丢失 |
| **#1000+** | 系统变量 | 读取机床坐标、模态、状态 |

### 常用赋值示例
```nc
#1=50.0       X坐标赋值
#2=-30.0      Z坐标赋值
#3=2.0        螺距/余量
#4=0.6495*#3  牙高计算
```
## 二、运算符号（数学计算）
| 符号 | 功能 | 示例 |
| :--- | :--- | :--- |
| + | 加法 | #5=#1+#2 |
| - | 减法 | #5=#1-#2 |
| * | 乘法 | **#5=#1*#2** |
| / | 除法 | #5=#1/#2 |
| SQRT | 开方 | #5=SQRT[#1] |
| ABS | 绝对值 | #5=ABS[#1] |
| SIN | 正弦 | #5=SIN[#1] |
| COS | 余弦 | #5=COS[#1] |
| ROUND | 四舍五入 | #5=ROUND[#1] |

## 三、条件判断（IF 语句）
```nc

IF [条件表达式] GOTO N

```
