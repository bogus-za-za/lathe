# G代码

## 01插补类

### G1直线插补

:::tip

代码格式 **G1 X(U)_ Z(W)_ F_**

:::


| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G1** | 直线插补模态指令 | 一旦指定，直到被同组的G代码（如G0, G2, G3）取代前一直有效。 | **是** |
| **X(U)_** | 目标点的 X 轴坐标 | 直径编程时指定直径值。 | 与Z二选一或同时 |
| **Z(W)_** | 目标点的 Z 轴坐标 | 指定轴向位置。 | 与X二选一或同时 |
| **F_** | **进给速度** | **这是G1的灵魂。** 指定刀具移动的速度。 | **是** |

:::tip

示例 假设毛坯为 Ø50mm 的棒料，现在要进行端面、外圆和锥面的粗精加工。

:::

### G2G3(圆弧插补)

:::tip

代码格式 **G2 X(U)_Z(W)_I_K_ F_** 或者 **G2 X(U)_Z(W)_R_F_**

:::

| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G2** | 顺时针圆弧插补模态指令 | 直到被同组的G代码（如G0, G1, G3）取代前一直有效。 | **是** |
| **G3** | 逆时针圆弧插补模态指令 | 直到被同组的G代码（如G0, G1, G2）取代前一直有效。 | **是** |
| **X(U)_** | 圆弧终点的 X 坐标 | 直径编程时指定直径值。 | **与Z二选一或同时** |
| **Z(W)_** | 圆弧终点的 Z 坐标 | 指定轴向位置。 | **与X二选一或同时** |
| **I_** | 圆心在 X 方向的增量 | **从圆弧起点到圆心的 X 方向增量（半径值）**，有正负。 | **与K和R二选一** |
| **K_** | 圆心在 Z 方向的增量 | **从圆弧起点到圆心的 Z 方向增量**，有正负。 | **与I和R二选一** |
| **R_** | 圆弧半径 | 指定圆弧的半径，有正负（正负表示圆弧大小）。 | **与I、K二选一** |
| **F_** | **进给速度** | 指定圆弧插补的进给速度。 | **是** |

G2/G3 选择速查表：

| 刀具位置 | 圆弧类型 | 通常使用 | 操作者视角判断 |
| :--- | :--- | :--- | :--- |
| 外圆车削 | 凸圆弧（外凸） | G3 | 逆时针 |
| 外圆车削 | 凹圆弧（内凹） | G2 | 顺时针 |
| 内孔车削 | 凸圆弧（内孔凸） | G2 | 顺时针 |
| 内孔车削 | 凹圆弧（内孔凹） | G3 | 逆时针 |

I、K方向判断图示
```plaintext

      Z轴
       ↑
       |    (I负,K正)   (I正,K正)
       |         ↖         ↗
       |          圆弧起点
       |         ↙         ↘
       |    (I负,K负)   (I正,K负)
       |
O——————|——————→ X轴

```

终点坐标：可以用绝对坐标（X, Z）或增量坐标（U, W）指定。

圆心指定方式（二选一）：
1. I、K 方式（矢量/圆心坐标法）：
   - I：从圆弧起点到圆心在X方向的矢量分量（半径值）。方向与X轴正方向相同时为正。
   - K：从圆弧起点到圆心在Z方向的矢量分量。方向与Z轴正方向相同时为正。
   - 公式：圆心坐标 = 起点坐标 + (I, K)
   - 优点：可以编程大于180°的圆弧（整圆只能用此方式）。

2. R 方式（半径法）：
   - 直接用R指定圆弧半径。
   - R值的正负决定圆弧大小：
        R > 0：圆心角 ≤ 180° 的圆弧（劣弧）
        R < 0：圆心角 > 180° 的圆弧（优弧）
   - 注意：不能用于整圆编程（需要分两段）。
   - 优点：编程简单，无需计算圆心坐标。

进给速度 F：与G1相同，由G98/G99决定单位。

## 02循环类

### G71 (外圆 / 内孔粗车复合循环)
:::tip

径向粗车循环 G71 U (背吃刀量) R (退刀量)<br/>
G71 P (起始行) Q (结束行) U (X 余量) W (Z 余量) F_

:::

| 代码 / 地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| G71 | 外圆 / 内孔粗车复合循环 | 自动分层切削，适配复杂轮廓 | 是 |
| **U_** | 粗车背吃刀量 | 半径值，无正负，如 U2.0 = 每次切 4mm 直径 | 第一行必选 |
| **R_** | 退刀量 | 切削后 X 向退刀量，常用 R0.5~1.0 | 第一行必选 |
| **P_** | 精车程序起始行号 | 轮廓程序开始行 | 第二行必选 |
| **Q_** | 精车程序结束行号 | 轮廓程序结束行 | 第二行必选 |
| **U_** | X 轴精车余量 | 直径值，外圆留 0.2~0.5，内孔留 0.2~0.5 | 第二行必选 |
| **W_** | Z 轴精车余量 | 常用 0.1~0.2 | 第二行必选 |
| **F_** | 粗车进给速度 | 粗车专用进给 | 第二行必选 |


**核心特点**
   - **自动分层切削**：只需写最终轮廓，机床自动计算分层
   - **适用场景**：余量较大、带圆弧 / 倒角 / 台阶的复杂外圆 / 内孔
   - **轮廓要求**：P-Q 之间轮廓必须单调增大 / 减小（无凹槽）
   - **优势**：复杂零件编程效率提升 80%，无需手动计算分层


<Tabs>
   <TabItem value="外圆粗车" label="外圆粗车" default>
   ``` nc

     T0101;
     M03 S1200;
     G0 X62.0 Z2.0; // 循环起点（大于毛坯直径）
     // G71第一行：背吃刀量2mm，退刀量1mm
     G71 U2.0 R1.0;
     // G71第二行：P10起始 Q20结束 X留0.3mm余量 Z留0.1mm
     G71 P10 Q20 U0.3 W0.1 F0.2;
     N10 G0 X20.0; // 轮廓起始行
     G1 Z0 F0.15;

     Z-15.0;// 外圆

     G1 X50.0 Z-30.0; // 锥度
     Z-50.0;N20 X60.0;
      // 轮廓结束行
     G70 P10 Q20;
       // 精车循环
     G0 X100.0 Z100.0;
     M30;
   ```
   </TabItem>

### G72（端面粗车复合循环）
:::tip

端面粗车循环格式G72 W (背吃刀量) R (退刀量)<br/>
G72 P (起始行) Q (结束行) U (X 精车余量) W (Z 精车余量) F_

:::

| 代码 / 地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| G72 | 端面粗车复合循环模态指令 | 沿 Z 轴分层切削，适合端面大余量、短粗零件 | 是 |
| W_ | 粗车背吃刀量 | Z 方向每次切削深度，半径值，无正负 | 第一行必选 |
| R_ | 退刀量 | 每层切削后 Z 向退刀量，常用 0.5~1.0 | 第一行必选 |
| P_ | 精车程序起始行号 | 轮廓程序开始行号 | 第二行必选 |
| Q_ | 精车程序结束行号 | 轮廓程序结束行号 | 第二行必选 |
| U_ | X 轴精车余量 | 直径值，外圆一般 0.2~0.5 mm | 第二行必选 |
| W_ | Z 轴精车余量 | 一般 0.1~0.2 mm | 第二行必选 |
| F_ | 粗车进给速度 | 端面粗车切削进给 | 第二行必选 |

**特点与适用场景**
  - **切削方向**：Z 向进刀，X 向车削，适合端面余量很大的盘类零件
  - **轮廓要求**：P～Q 之间轮廓必须单调变化，不能有内凹
  - 与 G71 区别：G71 车外圆 / 内孔（Z 长），G72 车端面（X 长）
<Tabs>

   N10 G0 Z-30.0; // 轮廓起始（从 Z 深处开始）

   Z0;

### G73(仿形粗车复合循环)

:::tip

代码格式 **G73 U(Δi) W(Δk) R(d)**<br/>
**G73 P(ns) Q(nf) U(Δu) W(Δw) F(f)**

:::

| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G73** | 仿形粗车复合循环模态指令 | 沿零件轮廓分层切削，适合成型毛坯 | **是** |
| **U(Δi)_** | X轴总切削余量 | 半径值，毛坯X方向总余量 | **是** |
| **W(Δk)_** | Z轴总切削余量 | Z方向总切削余量 | **是** |
| **R(d)_** | 粗车切削层数 | 分多少刀完成粗车，常用5~10 | **是** |
| **P(ns)_** | 精车程序起始行号 | 轮廓程序开始行 | **是** |
| **Q(nf)_** | 精车程序结束行号 | 轮廓程序结束行 | **是** |
| **U(Δu)_** | X轴精车余量 | 直径值，外圆留0.2~0.5mm | **是** |
| **W(Δw)_** | Z轴精车余量 | 常用0.1~0.2mm | **是** |
| **F(f)_** | 粗车进给速度 | 仿形粗车切削进给 | **是** |

 **核心特点**
1. **仿形切削**：刀具轨迹与零件轮廓完全一致，均匀去除余量
2. **适用场景**：**锻造毛坯、铸造毛坯、盘类零件、复杂圆弧轮廓**
3. **与G71区别**：G71只能车单调轮廓，G73可车任意复杂轮廓
4. **优势**：刀具受力均匀，延长刀具寿命，适合成型件加工

**编程注意事项**
  - 循环起点：必须在毛坯外侧，确保安全
  - 切削层数：根据余量大小设置，层数越多切削越平稳
  - 精车余量：外圆 U 为正，内孔 U 为负
  - 配合使用：粗车完成后必须紧跟 G70 精车循环

### G74（端面深孔 / 深槽钻削循环）
:::tip
端面深孔/断屑循环格式
**G74 R(退刀量)**<br/>
**G74 X(U)_ Z(W)_ P(X每次进刀) Q(Z每次进刀) R(退刀量) F_**
:::

| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G74** | 端面深孔 / 深槽断屑循环 | Z 向间歇切削，自动断屑、排屑 | **是** |
| **R_** | 退刀量 | 每次切削后 Z 向退刀断屑量 | 第一行必选 |
| **X(U)_** | 槽底 X 坐标 | 端面槽加工时使用，钻孔可省略或写 X0 | 可选 |
| **Z(W)_** | 孔底 / 槽底 Z 坐标 | 最终加工深度 | **是** |
| **P_** | X 方向每次切入量 | 端面槽用，钻孔一般写 P0 | 必选 |
| **Q_** | Z 方向每次进刀量 | 每次钻削深度，无正负 | 必选 |
| **F_** | 进给速度 | 钻削/切槽进给 | **是** |

 **特点与适用场景**
- 主要用于：**端面深孔钻削、端面宽槽、Z 向深槽**
- 自带断屑：切一点 → 退一点 → 再切，防止缠刀、扎刀
- 只做直线往复，不加工圆弧轮廓

### G75（外圆/内孔切槽循环）
:::tip
外圆/内孔切槽循环格式
**G75 R(退刀量)**<br/>
**G75 X(U)_ Z(W)_ P(X每次进刀) Q(Z每次进刀) R(退刀量) F_**
:::

| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G75** | 外圆/内孔沟槽循环 | X 向间歇切削，用于切槽、退刀槽、宽槽加工 | **是** |
| **R_** | 退刀量 | 每次切削后 X 向退刀断屑 | 第一行必选 |
| **X(U)_** | 槽底 X 坐标 | 槽底直径值 | **是** |

| **F_** | 切槽进给速度 | 切槽专用，进给不宜过大 | **是** |

 **特点与适用场景**
- 专门用于：**外圆退刀槽、密封槽、宽槽、内孔槽**
- 自动分层切入，自动断屑，保护槽刀
- 窄槽：Q0 只在 X 向分层切削
- 宽槽：Q≠0 实现 Z 向走刀，铣削式开槽

### G76(螺纹切削复合循环)

:::tip

代码格式 **G76 P(m)(r)(a) Q(最小切深) R(精车余量)**<br/>
**G76 X(U)_ Z(W)_ R(i) P(k) Q(Δd) F_**

:::

| 代码/地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| **G76** | 螺纹切削复合循环模态指令 | 自动分层车螺纹，支持外螺纹/内螺纹/锥螺纹 | **是** |
| **P(m)(r)(a)** | 螺纹综合参数 | m：精车次数(1-99) r：倒角量 a：螺纹角度 | **是** |
| **Q(Δdmin)** | 最小切入量 | 半径值，最后一刀最小切深，防止扎刀 | **是** |
| **R(d)** | 精车余量 | 半径值，一般0.05~0.1 | **是** |
| **X(U)_** | 螺纹底径坐标 | X：绝对坐标 U：增量坐标 | **是** |
| **Z(W)_** | 螺纹终点坐标 | Z：绝对坐标 W：增量坐标 | **是** |
| **R(i)** | 锥螺纹差值 | 圆柱螺纹R0，锥螺纹用半径差 | 否 |
| **P(k)** | 螺纹牙高 | 半径值，牙高=0.6495×螺距 | **是** |
| **Q(Δd)** | 第一刀切深 | 半径值，第一刀最大切削深度 | **是** |
| **F_** | 螺纹螺距 | 公制螺纹直接写螺距值 | **是** |

 **核心特点**
1. **自动分层切削**：只需写1行指令，机床自动计算分层，不用写多刀G32
2. **切削方式**：单边切削（刀刃受力小，不易崩刀），适合大螺距/强力螺纹
3. **支持类型**：外螺纹、内螺纹、圆柱螺纹、锥螺纹、公制/英制螺纹
4. **优势**：编程极简单，比G92更稳定、更适合批量生产

 **G76 P参数详解（三合一）**
**P m r a**（6位数字）
- **m**：精车次数（01~99），常用 01~03
- **r**：螺纹收尾倒角量（00~99），常用 00~10
- **a**：螺纹角度（60°/55°），公制写 60，英制写 55

示例：**P020560**
- 02 = 精车2次
- 05 = 收尾倒角量5
- 60 = 螺纹角度60°

**螺纹计算公式（必须牢记）**
- P螺距[第二行R值]
- 螺纹大径 =公称直径 - 0.15 × **螺距** 
- 牙高（半径值）= 0.6495 × **螺距** [第二行P值]
- 螺纹底径 = 大径 - 1.08 × **螺距** [第二行X值]

**实战编程要点**
  - **牙高** P 必须精确：直接用公式 0.6495× 螺距 计算
  - **第一刀切深** Q：钢件一般 0.3~0.5，铸铁 0.4~0.6
  - **最小切深** Q (min)：设 0.05，防止最后一刀空切或啃刀
  - **主轴转速**：车螺纹时转速不宜过高，S400~S800 最稳定
  - **收尾** r：普通螺纹用 05 即可，细长件可适当加大
  - **注意第二行PQ单位** 机床单位

## 04单一固定循环


:::tip

外圆车削循环 G90 X(U)_ Z(W)_ R_ F_端面车削循环 G94 X(U)_ Z(W)_ R_ F_

:::


| 代码 / 地址 | 含义 | 说明 | 是否必选 |
| :--- | :--- | :--- | :--- |
| G90 | 外圆 / 锥面单一循环 | 轴向 (Z 轴) 为主切削方向，车外圆、锥面 | 是 |
| G94 | 端面 / 斜端面单一循环 | 径向 (X 轴) 为主切削方向，车端面、斜端面 | 是 |
| X(U)_ | 循环终点 X 坐标 | 直径编程，U 为增量值 | 是 |
| Z(W)_ | 循环终点 Z 坐标 | W 为增量值 | 是 |
| R_ | 锥度差值 | 起点与终点的 X/Z 半径差，无锥度可省略 | 否 |
| F_ | 进给速度 | 循环切削进给率 | 是 |

核心特点
   - 一键简化编程：一次指令完成「切入→切削→退刀→返回」4 个动作
   - 适用场景：余量小、形状简单的外圆 / 端面切削
   - 模态特性：指定一次后持续生效，直到被 G0/G1/G2/G3 取代
   - 优势：比逐段写 G1 代码少 50% 以上，减少出错
