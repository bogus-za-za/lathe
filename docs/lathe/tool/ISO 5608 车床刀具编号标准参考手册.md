
# 标准编号系统结构
## 1.1完整刀杆编号格式
```TXT
□ □ □ □ - □□ □□ □□ - □□
│ │ │ │    │  │  │   └─ 制造商标记（可选）
│ │ │ │    │  │  └─ 刀尖半径
│ │ │ │    │  └─ 刀杆长度
│ │ │ │    └─ 刀杆截面尺寸
│ │ │ └─ 刀片固定方式
│ │ └─ 刀片精度等级
│ └─ 刀片后角
└─ 刀片形状
```

```MARKDOWN
## 刀杆型号：QZP 200500R-04

### 特殊说明：
切断/切槽刀有独立编号系统：
- **Q** - 切断刀类型
- **Z** - 刀片方向
- **P** - 刀杆截面形状
- **20** - 刀杆高度20mm
- **05** - 刀杆宽度5mm
- **00** - 长度代码
- **R** - 右手刀
- **04** - 刀片宽度4mm

### 应用场景：
- 外圆切断
- 窄槽加工
- 端面槽加工
```
